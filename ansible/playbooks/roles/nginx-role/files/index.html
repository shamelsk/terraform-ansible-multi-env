Multi-Environment Terraform-Ansible Project

This project demonstrates automated infrastructure provisioning and configuration management using Terraform and Ansible. Multiple environments are managed through Infrastructure as Code, enabling scalable, repeatable, and reliable deployments on AWS.

✅ Project Successfully Deployed

Project Overview

Terraform

Provisioned AWS infrastructure including EC2 instances, networking components, and environment-specific resources using Infrastructure as Code.

Ansible

Automated server configuration, package installation, service management, and application deployment through Ansible playbooks.

Multi-Environment

Separate development, testing, and production environments managed with reusable Terraform modules and Ansible inventories.

AWS Cloud

Cloud infrastructure hosted on AWS with automated provisioning and configuration workflows.
